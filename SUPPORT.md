# Support

`codex-healthkit` is an early open source project.

The maintainer cannot provide private support for every local Codex environment, but good reports are welcome.

## Where To Ask

- Bug reports: open a GitHub issue using the bug report template.
- Documentation problems: open a GitHub issue using the documentation template.
- Safety concerns: use the safety-boundary issue template, or follow [SECURITY.md](SECURITY.md) if the concern is sensitive.

## Before Sharing A Report

Please review the report yourself before posting it.

Do not share:

- credentials
- tokens
- cookies
- private project paths
- account identifiers
- raw session transcripts
- raw `codex doctor` output

If in doubt, redact more.

## What Helps

Useful support requests include:

- your operating system
- the exact `codex-healthkit` command you ran
- whether you used default mode or optional doctor mode
- the redacted summary output
- what you expected to happen
- what happened instead

## What This Project Does Not Support

This project does not provide:

- Codex account support
- API billing support
- quota or usage recovery
- account switching
- transcript analysis
- cleanup of local Codex data

For official Codex behavior, use OpenAI's official support and documentation.
