# v0.1.0-alpha.2

Source-only alpha release candidate for `codex-healthkit`.

Package version: `0.1.0-alpha.2`  
Git tag: `v0.1.0-alpha.2`

## Highlights

- Compare one explicit previous JSON report with current metadata using `--compare <previous-report.json>`.
- Keep comparison stateless: no automatic history and no telemetry.
- Clarify the opt-in `--with-codex-doctor` boundary and provider reachability possibility.
- Include only redacted doctor summary fields; raw official doctor output is not included.
- Add fixture-only responsive README visuals and a reproducible Social Preview candidate.
- Improve English and Japanese onboarding, mode selection, uninstall, and troubleshooting guidance.
- Link to a real-world Codex CLI update log using before-and-after checks.

## Install From The Pinned Tag

After the prerelease tag is published:

```bash
git clone --branch v0.1.0-alpha.2 --depth 1 \
  https://github.com/Ishikawa-Hidekazu/codex-healthkit.git
cd codex-healthkit
./bin/codex-healthkit --version
./bin/codex-healthkit check
```

Expected version:

```text
codex-healthkit 0.1.0-alpha.2
```

## Safety Boundary

- Default mode does not execute `codex`.
- Credentials, tokens, cookies, SQLite contents, and session transcript contents are not read.
- Previous-report comparison reads only one explicitly supplied `codex-healthkit` JSON report.
- Official `codex doctor --json` runs only with `--with-codex-doctor` and may perform provider reachability checks.
- Raw official doctor output is not included; only redacted summary fields are reported.
- No cleanup, deletion, daemon, dashboard, session recorder, telemetry, or automatic history is added.

Review reports before sharing because file sizes and counts still describe a local environment.

## Known Limits

- Source-only alpha; tested on macOS and Linux.
- Windows is not supported by this Bash implementation.
- Comparison mode requires `jq`.
- No PyPI or npm package, Homebrew formula, binary artifact, or manually attached release artifact.
- GitHub-generated source archives are the only release downloads.
