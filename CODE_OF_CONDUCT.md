# Code Of Conduct

`codex-healthkit` is a small open source project for practical, safety-minded diagnostics.

Everyone participating in this project is expected to communicate respectfully and constructively.

## Expected Behavior

- Be kind and direct.
- Assume good intent when possible.
- Stay focused on the issue or pull request.
- Be patient with people who are new to the project.
- Respect privacy and do not ask people to share secrets, tokens, private logs, or raw transcripts.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Publishing someone else's private information.
- Asking for or posting credentials, tokens, cookies, private transcripts, or private customer data.
- Repeated off-topic comments after maintainers ask to stop.
- Deliberately misleading security reports or exploit instructions in public threads.

## Scope

This code of conduct applies to project spaces, including issues, pull requests, discussions, and other public project communication.

## Reporting

If you see behavior that violates this code of conduct, contact the maintainer privately through:

https://ishikawa.co/contact/

For security vulnerabilities, follow [SECURITY.md](SECURITY.md) instead.

Maintainers may edit, hide, or remove comments, close threads, or block participants when needed to keep the project safe and usable.
